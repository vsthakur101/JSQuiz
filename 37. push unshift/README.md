## Example: Array Manipulation with `push` and `unshift`

```javascript
const arr = [1, 2];        // Initialize an array with elements 1 and 2
arr.push(3, 4);            // Adds elements 3 and 4 to the end of the array
arr.unshift(5, 6);         // Adds elements 5 and 6 to the start of the array
console.log(arr);          // Output the modified array
